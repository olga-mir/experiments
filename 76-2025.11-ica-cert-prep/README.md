# Intro

This folder contains two sets of practice scenarios to practice istio for Istio Certified Associate exam.

Both were generated with Claude Code without prior knowledge of the exam questions.

# Setup

BYO your own cluster with istio installed. Istio install and customisation are part of the exam curriculum https://github.com/cncf/curriculum/blob/master/ICA_Curriculum.pdf but they are not included in either sets.

Each set comes with its setup script and challenges in Markdown files.
