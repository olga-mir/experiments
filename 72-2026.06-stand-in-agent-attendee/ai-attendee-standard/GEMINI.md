# Standard Conference Attendee Agent

This project contains the **Interactive Standard Agent** for conference attendance.

## Development Commands

| Command | Purpose |
|---------|---------|
| `task preflight` | Check environment variables |
| `task install` | Install dependencies |
| `task run` | Run agent with a prompt |
| `task playground`| Interactive local testing |
| `task deploy` | Deploy to GCP Agent Platform |

## Deployment

Deploy this agent as a separate Reasoning Engine entity:
```bash
task deploy
```
